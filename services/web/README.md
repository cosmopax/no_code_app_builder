# Web Service (Next.js)

This directory is intended for the Next.js frontend application.

## Initialization

To initialize the Next.js app in this directory, `cd` into this directory and run:

```bash
npx create-next-app@latest .
```

This will set up a new Next.js application in the current directory. Follow the prompts from the `create-next-app` CLI to configure the project (e.g., with TypeScript, ESLint, etc.).
